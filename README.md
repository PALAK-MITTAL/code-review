🚀 AI Code Review & Analysis Platform

An AI-powered code analysis platform that automatically detects bugs, identifies code smells, and suggests optimization improvements using Large Language Models (LLMs).

✨ Features
🔍 Automated code analysis and bug detection
🤖 AI-driven code quality recommendations using Gemini API
⚡ Real-time feedback with syntax highlighting
📊 Structured and easy-to-understand review output
🌐 REST APIs for instant code review and analysis
📈 Scalable backend architecture

🏗 Architecture
User submits source code through the React frontend.
Backend API receives and processes the code.
Custom prompts are sent to Gemini API.
AI analyzes the code for:
Bugs
Performance issues
Best practice violations
Optimization opportunities
Structured feedback is returned and displayed with syntax highlighting.
